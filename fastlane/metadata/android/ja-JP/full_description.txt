Shojin App は、スマホで AtCoder の問題演習を快適に行うためのアプリです。

主な機能:
- 問題一覧・詳細表示
- WebView を用いたログイン
- Markdown / LaTeX によるコード・数式レンダリング
- テーマ切替 / ライセンスビューア

F-Droid 版の相違点:
- 自己アップデート機能は無効（FDROID_BUILD / ENABLE_SELF_UPDATE=false）
- APK のダウンロード/インストール UI を削除
- オンラインの Google Fonts を使用せず、同梱フォントまたはシステムフォントのみ使用

プライバシー:
- トラッキングや広告はありません。ネットワークアクセスは公開されている AtCoder ページのみに限定されます。
